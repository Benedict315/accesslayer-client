# Support

## Getting help

- For bug reports, use the bug report issue template.
- For scoped implementation work, use the maintainer task template.
- For docs gaps or setup confusion, use the documentation template.

## Before opening an issue

- Read [CONTRIBUTING.md](./CONTRIBUTING.md).
- Check the backlog in [docs/open-source/issue-backlog.md](./docs/open-source/issue-backlog.md).
- Confirm the problem belongs in the client repo and not the server or contracts repo.

## Project boundaries

This repository covers the frontend application only. Backend indexing, metadata, and auth belong in `accesslayer-server`. On-chain pricing, ownership, and fee rules belong in `accesslayer-contracts`.
